# Todo App

This repository contains a very small JavaScript todo application. The app is located in the `docs/` directory so it can be hosted using GitHub Pages.

## Running locally
Simply open `docs/index.html` in a browser to use the app.

## Deploying with GitHub Pages
1. Push the repository to GitHub.
2. In the repository settings, enable **GitHub Pages** and select `Deploy from a folder` with `docs` as the folder.
3. Once saved, GitHub Pages will host the site at `https://<username>.github.io/<repository>`.
